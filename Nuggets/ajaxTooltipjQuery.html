AJAX - Asynchronous JavaScript and XML

Background

When you enter an url in the browser, according to the client-server architecture, the browser (client) sends a request to the server and the server responds with the resources requested. This does a full page load. A full page load involves fetching the page and its associated recources like images, css, js files etc. Although most of these resources are cached by your browser the first time the page is requested (so that the consecutive requests are fast) there is still a full page load happening. Till the request is received, the page and its content is unavailable and the user is just sitting there and waiting (which is not good, coz no one like waiting).

There are many cases when you need to update only a part of the web page and not load the whole web page again (like header, footer etc). This is where AJAX comes to rescue.
